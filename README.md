# Astra Web

Astra's React web client, bootstrapped with Vite and trimmed down to the pieces
needed for a streaming-style movie experience.

## Stack

- React 19 + TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- Motion
- Lucide React
- Vitest + Testing Library
- Oxlint + Prettier

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run format
```

## Environment

Copy `.env.example` to `.env.local` when the app is ready to call a movie API.
